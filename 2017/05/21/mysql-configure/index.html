MySQL for Windows安装与配置(ZIP Archive版本)

发表于 2017-05-21

ZIP Archive版本的MySQL没有setup.exe，安装和配置过程需要在cmd下完成。

  • 配置my.ini文件
    解压文件夹假设位于E:\Program Files\下，进入mysql根目录（E:\Program Files\mysql-5.7.18-winx64），新建文件my.ini，配置如下：
     [client]
    port=3306
    [mysqld]
    basedir=E://Program Files//mysql-5.7.18-winx64
    datadir=E://Program Files//mysql-5.7.18-winx64//data
    skip-grant-tables
    port=3306
    character-set-server=utf8
    [mysql]
    default-character-set=utf8

    mysql-5.7.18-winx64替换成你所下载的mysql版本的文件夹名字，然后在根目录下新建文件夹取名为data.
  • 配置环境变量
    然后回到桌面，右击“计算机”->属性->高级系统设置->环境变量->新建系统变量
    变量名：MYSQL_HOME
    变量值：E:\Program Files\mysql-5.7.18-winx64
    变量值替换成你的解压文件夹位置。
    点击Path变量，在变量值最前面加上;%MYSQL_HOME%\bin;
    注意Path变量值的分号。
  • 手动安装
    以上工作完成之后打开cmd进入mysql的bin目录下
     $ cd “E:\Program Files\mysql-5.7.18-winx64”

    $ E:

    $ cd bin

    开始安装：
    $ mysqld –install MySQL –defaults-file=”E:\Program Files\mysql-5.7.18-winx64\my.ini

    安装过程需要管理员权限否则出现Install/Remove of the Service Denied!错误。提示Service successfully installed.即安装成功。
    在cmd下启动mysql服务：
    $ net start MySQL

    配置root用户密码：
    $ mysqladmin -u root -p password **

    到这里mysql就安装好了，下面介绍一些基本的命令操作
  • 基本命令
    打开mysql数据库：
    $ mysql
    退出mysql数据库： 
    $ quit;
    查看所有数据库： 
    $ show databases;
    查看数据库中所有表： 
    $ show tables;
